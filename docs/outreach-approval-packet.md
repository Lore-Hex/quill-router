# Outreach Approval Packet

Use one row per lead in the Google Sheet. Do not send until `status=approved`.

## Message Rule

Format:

```text
{{short_context_sentence}}

"{{relevant_quote}}"

Context: {{context_link}}
```

## Approval Checklist

- `relevant_quote` is copied from `docs/outreach-quote-bank.csv`.
- `context_link` is a TrustedRouter page from the quote-bank row.
- `approved_message` is short.
- `status` is `approved` before sending.
- `opt_out` is checked before sending.
- `sent_at` is blank before sending.
- For X and LinkedIn, the exact action is approved at action time before posting, commenting, reposting, connecting, or sending a DM.
