Apps building on TrustedRouter.

Apps that route through TrustedRouter can self-identify and appear here. Identification is opt-in by construction: an app shows up only if it sends a public title with its requests. Untitled traffic is bucketed anonymously — we never expose a workspace, key, or any prompt content.

  • ✓ Self-reported via the X-Title header (or HTTP Referer)
  • ✓ Counts and token volume only — never tenant, key, or prompt data
  • ✓ Anonymous "Direct" bucket for everything untitled
